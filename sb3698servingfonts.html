INTRODUCTION

Serving fonts is a method of making fonts avaible for users for new web pages or any other applications. There are many methods for using this.

methods used in serving fonts

  • Web font services like "Google fonts" and "Adobe fonts" are simpler methods to used for web pages.
  • Self-Hosting Fonts is something that they can choose their own host fonts for their own servers.
  • Font CDNs are used to host the fonts.
  • Data URI,directly converts CSS files using Data URIs,Where font sizes are encoded in Base64.
  • Local fonts or System fonts in this we can use specific fonts in your CSS to stand on ones system fonts.
